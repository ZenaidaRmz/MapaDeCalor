Considerações do projeto:

  • Este projeto é destinado a uma ONG que quer disponibilizar uma mapa geral da criminalidade com base em dados enviados pelo Governo do Estado (Secretaria de Segurança do RGS)
  • Em "map-canvas" será visualizado um Mapa de Intensidade das criminalidades conforme opções, filtros e tipos (abaixo do mapa).
    • Para o desenvolvimento do mapa deve-se utilizar algum tipo de interface da API do GoogleMaps, conforme exemplos abaixo:
      • https://developers.google.com/maps/documentation/javascript/examples/layer-heatmap
      • https://developers.google.com/maps/documentation/javascript/examples/layer-fusiontables-heatmap (* preferencialmente)
      • https://developers.google.com/maps/documentation/javascript/examples/circle-simple
  • Para fins de posicionamento (centralização) do Gráfico de Intensidade, utilizar a Latitude + Longitude de acordo com os bairros abaixo:
    • Castro Aleves: -31.349748,-54.09024
    • Centro: -31.326788,-54.103862
    • Floresta: -31.337294,-54.136563
    • Getúlio Vargas: -31.318313,-54.092965
    • Menino Deus: -31.329006,-54.125562
    • Pedras Brancas: -31.336157,-54.071174
    • Popular: -31.317124,-54.120211
    • São Jorge: -31.303677,-54.109311
    • São José: -31.335227,-54.112717
    • São Judas: -31.334765,-54.086154
    • Taruma: -31.304092,-54.133837
    • Zona Rural: -31.266446,-54.139853
  • Para montar o gráfico de intensidade, sempre utilizar a quantidade (COUNT) de ocorrências (linhas) de um determinado bairro/tipo (filtros).
    • Quanto mais associações (através do select), maior ou mais intenso o gráfico relacionado ao Mapa.
  • Banco de Dados real a ser utilizado está na pasta "ob_bd".
